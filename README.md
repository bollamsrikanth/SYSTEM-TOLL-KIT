# SYSTEM-TOLL-KIT
Windows administration toolkit developed for internal IT operations.

## Features

### Quick Cleanup
- Temp file cleanup
- Windows temp cleanup
- Prefetch cleanup

### Deep Repair
- DISM RestoreHealth
- SFC Scan
- CHKDSK Scan

### Network Repair
- Flush DNS
- Winsock Reset
- TCP/IP Reset

### System Report
- CPU Information
- RAM Information
- Disk Information
- System Information Export

### Disk Cleanup
- Recycle Bin Cleanup
- Windows Cleanup

### Full Maintenance
- Cleanup
- DISM
- SFC
- CHKDSK

## Usage

Run as Administrator.
